# Idontcare.com — The Interdimensional TV Tuner

A retro-surrealist web experience styled as a flickering 1970s/80s CRT television.

## Features
- **CRT Overlay**: Scanlines, flicker, and "fisheye" aesthetics.
- **Channel Dial**:
  - **CH 6 (The Static Void)**: Stare into the interactive noise.
  - **CH 17 (VCR Repair)**: Adjust the tracking slider to de-blur the signal.
  - **CH 42 (Deep Cut Lounge)**: A scrolling ticker of obscure TV trivia.
- **Easter Eggs**:
  - **The Prisoner Protocol**: Type `BE SEEING YOU` to deploy Rover.
  - **Max Headroom Intrusion**: Type `MAX` to trigger a glitch modal.
  - **Technicolor Tilt**: Click the "TILT" button for a 70s high-contrast palette.

## Technical Setup
1. **Install Dependencies**: `npm install`
2. **Development**: `npm run dev`
3. **Build**: `npm run build`

## Deployment to Cloudflare Pages
1. Connect your GitHub repository to Cloudflare Pages.
2. Build Command: `npm run build`
3. Build Output Directory: `dist`
